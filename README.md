Functions in Java 🚀

A simple Java repository created for learning and practicing Functions (Methods) in Java.
This repository contains beginner-friendly programs demonstrating:

Function creation
Return types
Passing arguments
Working with arrays in functions
Swapping numbers using functions
User input using Scanner

📂 Files Included

Functions.java - Different types of Java functions
Function2.java - Function returning a String
HelloFunction.java - Greeting user using functions
Swapnumbers.java - Swapping two numbers
Changevalue.java - Passing arrays to functions

This repository is made for educational purposes to help beginners understand how functions work in Java.
